# checklist-pre-pilot.md — Pre-Pilot Readiness Checklist

Run this before sending a course package to a school or pilot teacher.

---

## SECTION 1 — File Completeness

### 00_Admin
- [ ] `Course_Brief.md` — complete, no blank fields
- [ ] `course_brief.json` — valid JSON
- [ ] `status.json` — status is `READY_FOR_PILOT`
- [ ] No `REQUESTED_INPUT.md` present (would mean missing data)

### 01_Research
- [ ] `Research_Output.md` — min 5 factual points
- [ ] `Infographic_Ideas.md` — min 3 ideas
- [ ] `Podcast_Script.md` — 150–225 words

### 02_Curriculum
- [ ] `Syllabus.md` — learning outcomes + module breakdown
- [ ] `Lesson_Plans.md` — all modules using the flow template
- [ ] `Worksheets.md` — at least 1 worksheet
- [ ] `Rubric.md` — 3 criteria, 3 levels each
- [ ] `Instructor_Checklist.md` — teacher can run this without the author

### 03_Slides
- [ ] `Slide_Deck_Spec.md` — slide count within limits
- [ ] `Slide_Copy.md` — no paragraphs on slides
- [ ] `Visual_Assets_Brief.md` — color + font + style defined
- [ ] `Image_Prompts.md` — prompts for all visual slides

### 04_Interactive
- [ ] `Game_Spec.md` — complete (all 7 sections)
- [ ] `Build_Instructions.md` — teacher can set up without author
- [ ] `Test_Cases.md` — minimum 3 test cases

### 05_Marketing
- [ ] `Marketing_OnePager.md`
- [ ] `Video_Script.md` — reads in 60–90 seconds
- [ ] `Social_Assets.md` — 3 distinct posts

### 07_Export
- [ ] `QA_Checklist.md` — all 8 dimensions scored
- [ ] `Final_Checklist.md` — no blocking issues

---

## SECTION 2 — Pedagogic Readiness

- [ ] Every module has an active element (not passive)
- [ ] No theory block exceeds 15 minutes
- [ ] Final learner output is defined and achievable in the declared time
- [ ] At least one interactive element (game / quiz) is in the package
- [ ] Opening trigger exists in each module
- [ ] Real-world relevance addressed in at least 2 modules
- [ ] Age adaptation is visible (language, examples, task length)

---

## SECTION 3 — Production Handoff Readiness

- [ ] Canva designer can produce slides from `Slide_Deck_Spec.md` + `Visual_Assets_Brief.md`
- [ ] Teacher can run the game from `Build_Instructions.md` without calling anyone
- [ ] All student-facing tools are free and accessible in school
