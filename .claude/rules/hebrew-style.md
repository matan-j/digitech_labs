# hebrew-style.md — Hebrew Writing Standards

Apply to all Hebrew content output.

---

## REGISTER

| Content Type               | Register                        |
|----------------------------|---------------------------------|
| Student worksheets         | ידידותי, ברור, לא פורמלי מדי    |
| Lesson plans (teacher)     | מקצועי, ישיר, מדויק              |
| Slide copy                 | קצר, חזק, שאלות ישירות          |
| Marketing OnePager         | עסקי-חינוכי, ברור, ממוקד תועלת  |
| Video script               | נאמר בקול — טבעי, לא קריאה      |
| Social posts               | תלוי הפלטפורמה — ראה הנחיות     |

---

## WRITING RULES

1. **משפטים קצרים** — מקסימום 20 מילה למשפט בתוכן תלמידים
2. **אין פסקאות ארוכות** — עד 4 שורות בבלוק אחד
3. **כותרות חזקות** — לא "מבוא לנושא" אלא "מה גדול יותר מהאינטרנט?"
4. **פועל לפני עצם** — "בדוק, נתח, בנה" לפני הסבר
5. **אין ז'רגון בלי הגדרה** — כל מושג טכני מוגדר בפשטות בפעם הראשונה שמופיע
6. **עברית עסקית** — לא אקדמית, לא סלנגית

---

## FORMATTING RULES

- ✅ כותרות ברורות (H2, H3)
- ✅ בולטים קצרים (לא פסקאות בתוך בולטים)
- ✅ טבלאות לנתונים ולהשוואות
- ✅ הוראות ממוספרות לפעילויות
- ❌ אין Bold מוגזם — רק למונחים מרכזיים
- ❌ אין כותרת H1 בתוך קבצי תוכן — H2 כרמה הראשונה

## TERMS TO AVOID

| במקום             | השתמש ב           |
|-------------------|-------------------|
| לממש              | לבנות / ליצור     |
| אינטגרציה         | חיבור / שילוב     |
| להטמיע            | לשלב              |
| אוריינטציה        | כיוון             |
| פרמטרים           | פרמטרים (מותר אם מוגדר) |
| להתממשק           | לעבוד עם          |

## TERMINOLOGY CONSISTENCY

Always use these terms consistently across a course:

| Concept              | Hebrew Term to Use          |
|----------------------|-----------------------------|
| Big Data             | נתונים גדולים / ביג דאטה (ראשון בעברית) |
| Dataset              | מאגר נתונים                 |
| Visualization        | ויזואליזציה / תרשים / גרף  |
| Algorithm            | אלגוריתם                    |
| Machine Learning     | למידת מכונה                 |
| Student output       | תוצר התלמיד                 |
| Learning outcome     | מטרת למידה                  |
| Module               | מודול / יחידה               |

## SLIDE-SPECIFIC RULES

- כותרת שקף: עד 8 מילים
- בולט: עד 10 מילים
- שאלה לדיון: מסתיימת ב-?
- אין שקף שמתחיל ב-"בשיעור היום נלמד"
- כל שקף אינטראקטיבי מתחיל בפועל: "בדוק / בחר / שאל / חשוב"
